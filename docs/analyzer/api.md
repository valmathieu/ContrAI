# contrai-analyzer — API reference

`contrai-analyzer` is a Streamlit application rather than a library: there is no
unified `contrai_analyzer` namespace to document. The internal modules
(`bidding`, `engine`, `models`) live under `packages/contrai-analyzer/src/` and
are subject to change.

A proper API reference will be generated here once the analyzer exposes a
stable importable package.
