# contrai-core

Shared domain model for the ContrAI monorepo.

Will house `Card`, `Deck`, `Hand`, `Bid`, `Contract`, `Trick`, `Round` and related types currently in `contrai-engine`. Extracting them here is **phase 2** of the workspace migration — `contrai-engine`, `contrai-analyzer`, and `contrai-scraper` will then depend on `contrai-core` for shared model code.
